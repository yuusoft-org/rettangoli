## Overview

Available attributes:

* Scroll vertical: `sv`
* Scroll horizontal: `sh`


### Scroll horizontal


```html
<rtgl-view w="f" bgc="isu" p="m" d="h">
  <rtgl-view bgc="su" p="m" wh="100" d="h" g="m" sh>
    <rtgl-view bgc="such" wh="48""></rtgl-view>
    <rtgl-view bgc="such" wh="48""></rtgl-view>
    <rtgl-view bgc="such" wh="48""></rtgl-view>
    <rtgl-view bgc="such" wh="48""></rtgl-view>
  </rtgl-view>
</rtgl-view>
```
