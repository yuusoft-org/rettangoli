## Rettangoli Sites

`@rettangoli/sites` is the static-site engine used in Rettangoli projects.

It builds content from Markdown and YAML into static HTML under `_site/`, with templates, partials, and data composition.

### Quickstart

```bash
bunx rtgl sites build
bunx rtgl sites watch
```
