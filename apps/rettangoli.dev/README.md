
# Rettangoli Website

Website and user facing documentation for Rettangoli.

This website is built with @rettangoli/sites

## Development

### Build the site
```bash
bun run build
```

### Build and serve (development)
```bash
bun run dev
```

### Serve existing build
```bash
bun run serve
```

### Manual commands
```bash
# Build using CLI directly
rtgl sites build

# Serve the built site
bunx serve _site
```
