
# Rettangoli Website

Website and user facing documentation for Rettangoli.

This website is built with `rtgl sites` and imports its shared site assets from `@rettangoli/sitekit`.

## Development

### Build the site
```bash
bun run build
```

### Build and serve (development)
```bash
bun run dev
```

### Manual commands
```bash
# Build using CLI directly
rtgl sites build

# Watch and serve locally
rtgl sites watch
```
